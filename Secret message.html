Specified Content

This little program I'll be building is mainly getting four details of a person: First name, Last name, age and height. The details of that person will be stated below:

  • That person first letter in his/her first name should be equal to the second letter of that person's last name
  • That person's age shouldn't be above 20 years of age
  • That person's height in centimeters shouldn't surpass 172cm

If that person details equals or meets the requirement, the console should print out the Secret Message. And if he or she didn't qualify, then the console will be empty.
